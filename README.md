# Twitter Linked List Management System

A simple C++ project based on Linked Lists for BCA students.

## Features

### Singly Linked List
- Insert at Beginning
- Insert at End
- Delete Operations
- Traversal of Nodes
- Search Operations
- Node Counting

### Doubly Linked List
- Insert at Beginning
- Insert at End
- Delete Operations
- Forward Traversal
- Backward Traversal
- Search Operations
- Node Counting

## Project Files

- `MainLinkListFIle.cpp` → Main menu and program execution
- `singlyLinkListBYME.hpp` → Singly Linked List implementation
- `DoublyLinkListBYME.hpp` → Doubly Linked List implementation

## Technologies Used
- C++
- Object-Oriented Programming
- Linked List Data Structure

## How to Run

### Using g++ Compiler

```bash
g++ MainLinkListFIle.cpp -o app
./app
```
